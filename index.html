⚙️ COMO CONECTAR COM O GOOGLE PLANILHAS

Copie o código abaixo e cole no Apps Script da sua planilha (Extensões → Apps Script). Ele já cuida de somar pontos de nomes iguais e salvar fotos no Drive.

PASSO 1 — COLE ESTE CÓDIGO NO APPS SCRIPT
// ── CONFIGURAÇÃO ────────────────────────────────────── const PASTA_FOTOS_ID = "COLE_O_ID_DA_PASTA_DO_DRIVE_AQUI"; // Crie uma pasta no Drive, abra ela, copie o ID da URL function doPost(e) { const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Ranking"); const dados = JSON.parse(e.postData.contents); const nome = dados.nome.trim().toLowerCase(); const pontos = Number(dados.pontos); const tarefa = dados.tarefa; const fotoB64 = dados.foto || ""; // Salva foto no Drive (se enviada) let linkFoto = ""; if (fotoB64) { const bytes = Utilities.base64Decode(fotoB64.split(",")[1]); const blob = Utilities.newBlob(bytes, "image/jpeg", nome + "_" + Date.now() + ".jpg"); const pasta = DriveApp.getFolderById(PASTA_FOTOS_ID); const arq = pasta.createFile(blob); arq.setSharing(DriveApp.Access.ANYONE_WITH_LINK, DriveApp.Permission.VIEW); linkFoto = arq.getUrl(); } // Busca se o nome já existe na planilha const dados_planilha = sheet.getDataRange().getValues(); let linhaExistente = -1; for (let i = 1; i < dados_planilha.length; i++) { if (dados_planilha[i][0].toString().trim().toLowerCase() === nome) { linhaExistente = i + 1; // +1 porque sheets é 1-indexed break; } } if (linhaExistente > 0) { // Nome já existe → soma pontos na coluna B const ptAtual = sheet.getRange(linhaExistente, 2).getValue(); sheet.getRange(linhaExistente, 2).setValue(ptAtual + pontos); sheet.getRange(linhaExistente, 4).setValue(new Date()); // última atualização } else { // Nome novo → adiciona linha sheet.appendRow([dados.nome.trim(), pontos, tarefa, new Date(), linkFoto]); } return ContentService .createTextOutput(JSON.stringify({ status: "ok" })) .setMimeType(ContentService.MimeType.JSON); }
PASSO 2 — CRIE A ABA "Ranking" NA PLANILHA

Renomeie a primeira aba para Ranking e coloque estes cabeçalhos na linha 1:

A1: Nome | B1: Pontos | C1: Última Tarefa | D1: Atualizado em | E1: Foto
PASSO 3 — PUBLIQUE COMO WEB APP
  1. Clique em Implantar → Nova implantação → Web App
  2. Em "Executar como": Eu mesmo
  3. Em "Quem tem acesso": Qualquer pessoa
  4. Copie a URL gerada (começa com https://script.google.com/macros/s/.../exec)
PASSO 4 — COLE A URL AQUI NESTE ARQUIVO HTML

Neste mesmo arquivo, procure a linha abaixo e substitua pelo link real:

const SHEET_URL = "COLE_SUA_URL_DO_APPS_SCRIPT_AQUI";

✅ Pronto! Cada envio vai: somar pontos para nomes iguais, salvar foto no Drive e atualizar o ranking automaticamente.
